## Product listing data

To display the product listing data, do one of the following:

- Use `products.json` and other necessary data provided in the starter files. Feel free to seed your database with this data.
- Fetch the data from our E-commerce product listing HTTP API.

## E-commerce product listing API

```
GET https://www.greatfrontend.com/api/projects/challenges/e-commerce/products
```

### Parameters

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `page` | `number` | No | Page number to retrieve. Defaults to 1. |
| `per_page` | `number` | No | Number of items per page. Defaults to 9. |
| `collection` | `Array<string>` | No | Collections to filter; Valid values include `latest` and collection IDs. |
| `category` | `Array<string>` | No | Categories to filter; Valid values include category IDs. |
| `color` | `Array<string>` | No | Variant colors to filter. |
| `rating` | `Array<number>` | No | Ratings to filter. |
| `sort` | `string` | No | Field to sort on; Valid values: `created`, `rating`, `popularity`, `price`. |
| `direction` | `string` | No | Sorting direction; Valid values: `asc`, `desc`; Default: `desc`. |

**Note**: To specify values fo `Array`-type parameters, repeat the query parameter:

```
https://www.greatfrontend.com/api/projects/challenges/e-commerce/products?color=black&color=white
```
