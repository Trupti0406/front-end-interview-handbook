In this challenge, you will build a versatile badge component as part of a design system. These badges will be used across various challenges and apps found within the platform, providing consistency throughout the user interface.

### Implementation requirements

- **Design fidelity**: Aim to follow the design as closely as possible. All elements in the design should be present, using the specified text color, font size, font weight, spacing, dimensions, etc.
- **Cross-browser compatibility**: Check that your solution works for major browsers including Chrome, Firefox, and Safari.

### Component properties

- **Colors**: Neutral, Error, Warning, Success, and Brand
- **Sizes**: Small, Medium, Large
