## Collections grid data

To display the collections grid data, do one of the following:

- Use `collections.json` provided in the starter files. Feel free to seed your database with this data.
- Fetch the data from our E-commerce collections HTTP API.

## E-commerce collections API

```
GET https://www.greatfrontend.com/api/projects/challenges/e-commerce/collections
```

### Parameters

No parameters.

### Returns

A list of e-commerce product collections.
