## Product details data

To display the product details data, do one of the following:

- Use `products.json` and other necessary data provided in the starter files. Feel free to seed your database with this data.
- Fetch the data from our E-commerce product details HTTP API.

## E-commerce product details API

```
GET https://www.greatfrontend.com/api/projects/challenges/e-commerce/products/:productId
```

Example: https://www.greatfrontend.com/api/projects/challenges/e-commerce/products/voyager-hoodie

Other product IDs include `urban-drift-bucket-hat`, `tangerine-mini-tote`, and `elemental-sneakers`. The full list of products can be found in the starter files.

### Parameters

No parameters.

### Returns

A product's details which include the name, category, collection, available colors, sizes, images, inventory (stock availability), rating, and total number of reviews.
