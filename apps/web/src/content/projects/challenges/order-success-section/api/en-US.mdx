## Order data

To display the order data, do one of the following:

- Build your own APIs that read order data from your own database using the provided products data.
- Use `sample-order.json` provided in the starter files. Feel free to seed the "Orders" table in your database with this data.
- Fetch sample order data from our E-commerce order HTTP API.

## E-commerce order API (sample data)

```
GET https://www.greatfrontend.com/api/projects/challenges/e-commerce/order-sample
```

**Warning**: This API always returns the same data as is not meant to be used in functional applications that should show the viewer's completed order.

This API is still useful in the following situations:

- If you only want to practice the front end aspects of fetching order data and displaying it.

No parameters.
