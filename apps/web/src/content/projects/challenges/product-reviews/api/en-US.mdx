## Product reviews data

To display the product reviews data, do one of the following:

- Use `reviews.json` and other necessary data provided in the starter files. Feel free to seed your database with this data.
- Fetch the data from our E-commerce product reviews HTTP API.

## E-commerce product reviews API

```
GET https://www.greatfrontend.com/api/projects/challenges/e-commerce/products/:productId/reviews
```

Example: https://www.greatfrontend.com/api/projects/challenges/e-commerce/products/voyager-hoodie/reviews

Other product IDs include `urban-drift-bucket-hat`, `tangerine-mini-tote`, and `elemental-sneakers`. The full list of products can be found in the starter files.

### Parameters

| Name       | Type     | Required | Description                               |
| ---------- | -------- | -------- | ----------------------------------------- |
| `page`     | `number` | No       | Page number to retrieve. Defaults to 1.   |
| `per_page` | `number` | No       | Number of items per page. Defaults to 12. |
| `rating`   | `number` | No       | Filter only reviews of this rating.       |
