## Useful keyboard shortcuts

Keyboard shortcuts help you achieve more with fewer keystrokes, improving your productivity and help you code faster. These are the most useful shortcuts for working on challenges.

### File

- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>S</kbd>: Saves file
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>S</kbd>: Save all
  files

### Edit

- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Z</kbd>: Undo
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>Z</kbd>: Redo
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>X</kbd>: Cuts line or selection
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>C</kbd>: Copy
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>V</kbd>: Paste
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>K</kbd>: Delete line
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>F</kbd>: Search within file
- <kbd>Alt</kbd> + <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>F</kbd>: Replace within
  file
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>F</kbd>: Search
  within workspace / all files
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>H</kbd>: Replace
  within workspace / all files
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>/</kbd>: Toggle line comment

### Selection

- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>A</kbd>: Selects all lines in file
- <kbd>Alt</kbd> + <kbd>Shift</kbd> + <kbd>Up</kbd>/<kbd>Down</kbd>: Copy line
  up/down
- <kbd>Alt</kbd> + <kbd>Up</kbd>/<kbd>Down</kbd>: Move line up/down
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>D</kbd>: Select the next occurrence of a
  selected text
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>L</kbd>: Select all
  occurrences within file
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>L</kbd>: Select line

### Others

- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>E</kbd>: Open
  explorer in sidebar
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>F</kbd>: Open search
  in sidebar
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd>: Open command
  palette
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>`</kbd>: Toggle integrated terminal

## Command palette

There are thousands of available keyboard shortcuts and they can be hard to remember. An alternative approach is to use the "Command palette" to quickly find and apply commands. It can be opened by pressing <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd>.

Simply type what you want to achieve, if a command is supported, it will show up and you can execute it. Some of the most common and useful commands:

- Transform text (many options including uppercase, lowercase, snake case, etc).
- Splitting the editor window left/right/up/down.
- Format document.
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>P</kbd>: Opens the Quick Open dialog to
  quickly search and open a file.
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>O</kbd>: Navigate to
  a symbol within a file.
- Markdown: Open preview.

## Useful extensions

VS Code extensions are additional features that can be installed to enhance the functionality of the editor.

We recommend installing the following VS Code extensions, or finding the equivalents for your IDE:

- [JavaScript and TypeScript Nightly](https://marketplace.visualstudio.com/items?itemName=ms-vscode.vscode-typescript-next)
- [GitLens – Git supercharged](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens)
- [Prettier – Code formatter](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)
- [TypeScript Importer](https://marketplace.visualstudio.com/items?itemName=pmneo.tsimporter)
- [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)
- [Stylelint](https://marketplace.visualstudio.com/items?itemName=stylelint.vscode-stylelint)
- [Code Spell Checker](https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker)
- [Tailwind CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss)
- [Optional: Tailwind Docs](https://marketplace.visualstudio.com/items?itemName=austenc.tailwind-docs)

There are thousands of extensions available in the VS Code Marketplace, and you can install them easily within the editor by pressing <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>E</kbd> to open the extensions panel within the side bar.
