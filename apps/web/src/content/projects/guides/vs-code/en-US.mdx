## Useful keyboard shortcuts

Keyboard shortcuts help you achieve more with fewer keystrokes, improving your productivity and help you code faster.

We've selected the most useful shortcuts when working on challenges from the list of VS code keyboard shortcuts for [Windows](https://code.visualstudio.com/shortcuts/keyboard-shortcuts-windows.pdf) and [macOS](https://code.visualstudio.com/shortcuts/keyboard-shortcuts-macos.pdf).

### General

- <kbd>Shift</kbd> + <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>P</kbd>: Show Command
  Palette

### Basic editing

- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>X</kbd>: Cut line (empty selection)
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>C</kbd>: Copy line (empty selection)
- <kbd>Alt</kbd> + <kbd>Up</kbd>/<kbd>Down</kbd>: Move line up/down
- <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>Up</kbd>/<kbd>Down</kbd>: Copy line
  up/down
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>K</kbd>: Delete line
- <kbd>Shift</kbd> + <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>\\</kbd>: Jump to
  matching bracket
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>]</kbd>/<kbd>[</kbd>: Indent/outdent
  line
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>/</kbd>: Toggle line comment

### Multi-cursor and selection

- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>L</kbd>: Selects current line
- <kbd>Shift</kbd> + <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>L</kbd>: Select all
  occurrences of current selection
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>F2</kbd>: Select all occurrences of
  current word

### Search and replace

- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>F</kbd>: Find
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>D</kbd>: Add selection to next Find
  match
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>L</kbd>: Select line

### Rich languages editing

- <kbd>F12</kbd>: Go to Definition
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>.</kbd>: Quick fix
- <kbd>F2</kbd>: Rename symbol

### Navigation

- <kbd>Ctrl</kbd> + <kbd>G</kbd>: Go to line
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>P</kbd>: Go to file

### Editor management

- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>\\</kbd>: Split editor
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>1</kbd>/<kbd>2</kbd>/<kbd>3</kbd>: Focus
  into 1st, 2nd, 3rd editor group

### File management

- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>S</kbd>: Saves file
- <kbd>Shift</kbd> + <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>S</kbd>: Save all
  files

### Display

- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>B</kbd>: Toggle sidebar visibility
- <kbd>Shift</kbd> + <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>E</kbd>: Show
  Explorer
- <kbd>Shift</kbd> + <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>F</kbd>: Show Search
- <kbd>Shift</kbd> + <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>H</kbd>: Replace in
  files
- <kbd>Shift</kbd> + <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>V</kbd>: Open
  Markdown preview

### Integrated terminal

- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>`</kbd>: Toggle integrated terminal

## Command palette

There are thousands of available keyboard shortcuts and they can be hard to remember. An alternative approach is to use the "Command palette" to quickly find and apply commands. It can be opened by pressing <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>P</kbd>.

Simply type what you want to achieve, if a command is supported, it will show up and you can execute it. Some of the most common and useful commands:

- Transform text (many options including uppercase, lowercase, snake case, etc).
- Splitting the editor window left/right/up/down.
- Format document.
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>P</kbd>: Opens the Quick Open dialog to
  quickly search and open a file.
- <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>O</kbd>: Navigate to
  a symbol within a file.
- Markdown: Open preview.

## Useful extensions

VS Code extensions are additional features that can be installed to enhance the functionality of the editor.

We recommend installing the following VS Code extensions, or finding the equivalents for your IDE:

- [JavaScript and TypeScript Nightly](https://marketplace.visualstudio.com/items?itemName=ms-vscode.vscode-typescript-next)
- [GitLens – Git supercharged](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens)
- [Prettier – Code formatter](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)
- [TypeScript Importer](https://marketplace.visualstudio.com/items?itemName=pmneo.tsimporter)
- [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)
- [Stylelint](https://marketplace.visualstudio.com/items?itemName=stylelint.vscode-stylelint)
- [Code Spell Checker](https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker)
- [Tailwind CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss)
- [Tailwind Docs](https://marketplace.visualstudio.com/items?itemName=austenc.tailwind-docs)

There are thousands of extensions available in the VS Code Marketplace, and you can install them easily within the editor by pressing <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>X</kbd> to open the extensions panel within the sidebar.
