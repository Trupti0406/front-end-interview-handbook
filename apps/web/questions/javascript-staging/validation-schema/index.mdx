Implement basic version of validation schema like [Yup](https://github.com/jquense/yup).

```js
let schema = yup.object().shape({
  name: yup.string().required(),
  age: yup.number().required().positive().integer(),
});

schema.isValid({
  name: 'jimmy',
  age: 24,
});
```
