## Test Cases

- Adding items
  - New items can be added to their respective lists.
  - New items are added to the bottom of the lists.
- "Transfer selected items" buttons
  - Button is disabled when no items are selected in the source list.
  - Upon clicking, selected items are transferred from the source to destination list.
- "Transferring all items" buttons
  - Button is disabled when the source list is empty.
  - Upon clicking, all items from the source list are transferred to the destination list.
- Transferred items are added to the bottom of their new lists.
