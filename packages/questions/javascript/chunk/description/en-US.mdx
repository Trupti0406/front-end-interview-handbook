## Lodash's Chunk method

Implement a function `chunk` that emulates the functionality of the Lodash utility function `_.chunk(array, [size=1])`. The function splits the input array into smaller arrays of length `size` and returns them within a **new** array. The function should not modify the original input array.

```js
chunk(array, [(size = 1)]);
```

- `array`: The array to split.
- `size`: The chunk size. A positive integer.

## Examples

```js
chunk(['a', 'b', 'c', 'd']); // => [['a'], ['b'], ['c'], ['d']]
chunk([1, 2, 3, 4], 2); // => [[1, 2], [3, 4]]
```

If the length is not divisible by the `size`, the final chunk will contain the remaining elements.

```js
chunk(['a', 'b', 'c', 'd'], 3); // => [['a', 'b', 'c'], ['d']]
```

The function should return an empty array if the `array` argument is empty.
