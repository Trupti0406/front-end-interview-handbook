Implement a function `clamp(number, lower, upper)` to clamp a number within the inclusive `lower` and `upper` bounds.

## Arguments

1. `value` _(number)_: The number to clamp.
1. `lower` _(number)_: The lower bound.
1. `upper` _(number)_: The upper bound.

## Returns

_(number)_: Returns the clamped number.

## Example

```js
clamp(-10, -5, 5); // => -5
clamp(10, -5, 5); // => 5
```
