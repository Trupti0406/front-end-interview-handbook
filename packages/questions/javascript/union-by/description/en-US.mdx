---
title: Union By
excerpt: Implement a function that creates an array of unique values, in order, from all given arrays.
---

Implement a function `unionBy(array)` that creates an array of unique values, in order, from all given arrays and accepts `iteratee` which is invoked for each element of each arrays to generate the criterion by which uniqueness is computed. The iteratee is invoked with one argument.

## Arguments

1. `iteratee` _(Function)_: The iteratee invoked per element
1. `[arrays]` _(...Array)_: The arrays to inspect

## Returns

_(Array)_: Returns the new array of combined values.

## Examples

```js
unionBy((value: any) => value, [2], [1, 2]); // => [2, 1]
unionBy(Math.floor, [2.1], [1.2, 2.3]); // => [2.1, 1.2]
unionBy('x', [{ x: 1 }], [{ x: 2 }, { x: 1 }]); // => [{ 'x': 1 }, { 'x': 2 }]
```
